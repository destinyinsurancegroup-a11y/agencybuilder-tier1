
# Agency Builder CRM — Tier 1 (Rebuild)

This is a deployable Tier 1 version of the Agency Builder CRM with:
- Dashboard + My Numbers tab
- Laravel 10 + Docker support
- PostgreSQL ready
- Beige/black/gold UI theme

To deploy:
1. Upload this ZIP to DigitalOcean App Platform.
2. Select Dockerfile as the build source.
3. Set DB_* environment variables.
4. Deploy. The app will auto-migrate and run.
